# the-erranza
A full stack web application that allows users to list, discover and book safari travels and destinations, short and long term rentals, hotel, list and rent cars, sell & buy touristic items, offer and rent services and also listing holiday homes for sale.


## Features
- Userauthenticattion (sign up, login, and logout)
- Safari listings with images and descriptions
- Hotels and stays listing with images and descriptions
- Car rentals lists and rentals
- Market place - buy and sell items
- Servce Directory - offer and deliver services. Hire experts
- Tourist attractionsm adn acativities taking place in specific places
- Searach and filter by location, price and amenities  #####
- Interactive map intergaration
- Booking system with calendar availability
- User Dashboard for managing listings and reservations
- Responsive design for mobile and desktop


----
##  Tech Stack
- **Frontend::** React/Next.js, TailwindCSS,
- **Backend:** Laravel (API)
- **Database:** Supabase, MongoDB
- **Authentication:** Firebase, JWT, bcrypt
- **CloudHosting:** AWS, GoogleCloud and Vercel(frontend)
- **Real-time:** Socket.io (for live tracking like Uber)
- **Maps:** Leeaflet.js / Google Maps API
